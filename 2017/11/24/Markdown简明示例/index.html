Html锚点页内跳转

实际语法比较简单，在需要跳转的位置添加锚点，语法如下：
<span id="jump">跳转到的地方</span>

在需要点击跳转的位置，使用上面的id，格式类似超链接的形式：
[点击跳转](#jump)

这样就实现了上述功能。
